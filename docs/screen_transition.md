## 画面遷移図
```mermaid
graph TD

  %% トップ
  A[トップページ<br>投稿一覧] 
  A --> B[新規登録]
  A --> C[ログイン]
  A --> D[投稿詳細]
  A --> E[新規投稿]
  A --> U[ユーザーマイページ<br>投稿一覧]
  A --> Q[プロフィール表示]
  A --> H[コラボ一覧<br>（送信・受信）]

  %% 投稿詳細からの遷移
  D --> F[コメント投稿（同ページ内）]
  D --> G[コラボ申請（同ページ内）]

  %% 認証後の遷移
  B --> A
  C --> A

  %% 投稿関連
  E --> A
  D --> P[投稿編集]
  P --> A

  %% コラボ関連
  H --> D
  H --> T[チャットルーム]

  %% プロフィール（追加＆修正）
  U --> Q
  Q --> U
  Q --> R[プロフィール編集]
  R --> Q

  %% チャット関連
  A --> S[チャット一覧]
  S --> T

  ```
